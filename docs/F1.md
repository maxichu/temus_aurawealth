# F1 - REPL AI Chat

## Feature Description

A terminal-based (REPL) chat interface that communicates with an LLM-backed
AI financial advisor. Supports multi-turn conversation and graceful exit.

Implementation: repl.py — runs in the terminal, calls chat() from lib/llm.py.

---

## Test Case 1 — Multi-turn financial conversation with context retention

Scenario: User asks a financial question, then follows up with a related question.
The assistant must retain context from the first turn.

 Steps:
 1. Run python repl.py — Welcome message and prompt appear
 2. Type "What is a good ETF for beginners?" — AI responds with ETF recommendations
 3. Type "Why do you recommend that specific fund?" — AI references previous response

Expected: The second response references the first (context maintained via messages[] list).

Actual: PASS — conversation history is appended after each turn and included in every
subsequent LLM API call.

Status: PASS

---

## Test Case 2 — Graceful exit via quit command

Scenario: User finishes a conversation and exits cleanly.

 Steps:
 1. Run python repl.py — Program starts normally
 2. Type "What is dollar-cost averaging?" — AI responds
 3. Type "quit" — Program prints "Goodbye!" and exits

Expected: No traceback. Clean exit with code 0.

Actual: PASS — the while loop breaks on "exit" or "quit" (case-insensitive),
and print("Goodbye!") is called before termination.

Status: PASS

---

## Test Case 3 — Empty input handling

Scenario: User presses Enter without typing, then continues normally.

 Steps:
 1. Run python repl.py — Program starts
 2. Press Enter (empty input) — Program re-prompts without calling the LLM
 3. Type "What is a portfolio?" — AI responds normally

Expected: Empty input does not trigger an API call or cause an error.

Actual: PASS — "if not user_input: continue" skips empty strings gracefully.

Status: PASS
